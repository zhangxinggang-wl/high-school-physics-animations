# 高中物理人教版教学动画合集

## 简介

本合集收录了 **105 个高中物理互动教学动画**（HTML 格式），覆盖人教版必修 + 选择性必修共六册教材，所有动画可直接在浏览器中打开使用。

动画内容涵盖力学、电磁学、热学、光学、近代物理等高中物理核心领域，适合教师课堂教学演示或学生自主探究学习。

## 使用方法

1. 下载或克隆本仓库到本地
2. 进入对应章节文件夹
3. **直接用浏览器打开任意 HTML 文件**即可观看互动动画

无需安装任何软件或插件，Chrome / Edge / Firefox 等主流浏览器均可正常打开。
